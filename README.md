## Description 
This site is a CMS-style blog site where developers can publish their blog posts and comment on other developersâ€™ posts as well.

## Installation
This uses Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication. The following were installed for this application:
- Node.Js
- Express
- Sequelize
- Postgres
